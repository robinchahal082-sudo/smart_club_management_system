Smart Club Management System

📌 Project Overview

Smart Club Management System is a web-based application developed to manage college/university clubs efficiently.
The system helps administrators, faculty coordinators, club leaders, and students manage club activities, events, memberships, announcements, and participation records digitally.

---

🚀 Features

Admin Module

- Manage students and faculty
- Create and manage clubs
- Assign club leaders and coordinators
- Monitor overall activities

Club Management

- Create new clubs
- Update club information
- Manage club members
- Assign leadership roles

Event Management

- Create and organize events
- Event registration system
- Track participation records
- Event approval workflow

Student Module

- Join clubs
- Register for events
- View announcements
- Track participation history

Authentication & Security

- Login and registration system
- Role-based access control
- Secure session management
